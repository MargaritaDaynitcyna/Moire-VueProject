# Moire-VueProject
Разработка интернет-магазина Moire (премиальное бельё) на Vue.js с использованием REST API.


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
